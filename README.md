# Linux Administration Lab

## Цели:
- Практика администрирования Linux серверов
- Настройка веб-сервисов (Django + Nginx + PostgreSQL)
- Реализация мониторинга и алертинга
- Автоматизация рутинных задач
- Практика резервного копирования

## Структура:
- `/scripts/` - скрипты автоматизации на Python и Bash
- `/configs/` - конфигурационные файлы сервисов
- `/docs/` - документация и заметки
- `/backups/` - скрипты и конфиги для бэкапов

## Стек:
- Ubuntu 24.04 LTS
- Django 4.x
- PostgreSQL 15
- Nginx
- Prometheus + Grafana
- Docker
